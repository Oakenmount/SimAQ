# SimAQ

**Status:** This repository is currently private and the code is not publicly available.  
The code will be released **upon request from the editor or reviewers** as part of the peer review process.

## About
This repository is associated with the manuscript:

> *SimAQ: Mitigating Experimental Artifacts in Soft
X-Ray Tomography using Simulated Acquisitions*  
> Submitted to *IEEE Transactions on Medical Imaging*, August 2025.

The repository will contain the full source code and documentation necessary to reproduce the results reported in the manuscript once it is made public.

## Availability
At this stage, the repository serves as a placeholder link for the manuscript submission.  
The code will remain confidential until the journal's editorial process requires or permits public release.
